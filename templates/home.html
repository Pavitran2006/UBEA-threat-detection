About the Project

UEBA (User and Entity Behavior Analytics) is an advanced cybersecurity solution designed to detect abnormal user activities inside an organization. The system analyzes login behavior, device fingerprints, IP address changes, and login patterns using Machine Learning algorithms.

  • Insider threats detection
  • Account compromise protection
  • Device fingerprinting
  • Location anomaly detection
  • Data exfiltration prevention

Key Features

AI Anomaly Detection

Uses Isolation Forest machine learning algorithm to detect abnormal user behavior in real-time.

Real-Time Risk Scoring

Each user is assigned a dynamic risk score based on activity patterns and behavioral analysis.

Device Fingerprinting

Tracks login devices to detect suspicious new devices attempting to access the system.

Login Behaviour Analytics

Analyzes login time, IP address, location, and frequency to identify anomalies.

Threat Intelligence Dashboard

Admin dashboard shows alerts, risk levels, and suspicious activity in real-time.

Admin Feedback Loop

Security admins can mark alerts as SAFE or THREAT to retrain the ML model.
