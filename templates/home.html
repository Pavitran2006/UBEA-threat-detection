About the Project

UEBA (User and Entity Behavior Analytics) is an advanced cybersecurity solution designed to detect abnormal user activities inside an organization. The system analyzes login behavior, device fingerprints, IP address changes, and login patterns using Machine Learning algorithms.

  • Insider threats
  • Account compromise
  • Suspicious login patterns
  • Unauthorized access attempts
  • Data exfiltration risks

Key Features

AI Anomaly Detection

Uses Isolation Forest machine learning algorithm to detect abnormal user behavior.

Real-Time Risk Scoring

Each user is assigned a dynamic risk score based on activity patterns.

Device Fingerprinting

Tracks login devices to detect suspicious new devices.

Login Behaviour Analytics

Analyzes login time, IP address, location, and frequency.

Threat Intelligence Dashboard

Admin dashboard shows alerts, risk levels, and suspicious activity.

Admin Feedback Loop

Security admins can mark alerts as SAFE or THREAT to retrain the ML model.

How the System Works

  1. User login activity is captured.
  2. System processes activity data.
  3. Machine learning model analyzes patterns.
  4. Anomalies are detected.
  5. Risk score is calculated.
  6. Admin dashboard displays alerts.
